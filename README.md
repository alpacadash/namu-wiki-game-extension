# 나무위키 게임 확장 프로그램

나무위키 게임은 나무위키 문서를 따라 이동하며 목표 문서까지 가장 빠르게 도달하는 멀티플레이 게임용 크롬 확장 프로그램입니다.

## 주요 기능

- 방 생성 및 참가
- 출발 문서 / 도착 문서 설정
- 실시간 순위 및 진행 상태 표시
- 인게임 HUD 표시
- 1위 도착 경로 공개
- 다시하기 / 로비 복귀

## 필요한 권한

- `storage`: 방 정보와 현재 세션 상태 저장
- `tabs`: 게임 시작 시 나무위키 탭 열기
- 호스트 권한: 나무위키 문서 페이지에서 HUD 표시 및 게임 진행 감지

## 동작 방식

확장 프로그램은 나무위키 문서 페이지에서만 동작하며, 사용자가 문서 링크를 따라 이동하는 진행 상황을 바탕으로 게임 상태를 표시합니다.
