# 나무위키 게임 확장 프로그램 개인정보처리방침

최종 수정일: 2026-03-28

## 1. 개요

나무위키 게임은 나무위키 문서를 탐색하며 목표 문서까지 도달하는 멀티플레이 게임용 크롬 확장 프로그램입니다. 이 문서는 확장 프로그램이 어떤 데이터를 처리하고, 어떤 목적으로 사용하는지 설명합니다.

## 2. 처리하는 정보

이 확장 프로그램은 게임 기능 제공을 위해 다음 정보를 처리할 수 있습니다.

- 사용자가 입력한 닉네임
- 방 코드, 플레이어 식별자, 현재 방 상태, 현재 라운드 정보
- 현재 보고 있는 나무위키 문서 제목 및 게임 진행 경로 정보

이 확장 프로그램은 다음 정보를 수집하지 않습니다.

- 실명, 주소, 이메일 주소, 전화번호
- 금융 정보 또는 결제 정보
- 건강 정보
- 정밀 위치 정보
- 개인 메시지, 이메일, 문자 내용

## 3. 정보 사용 목적

처리된 정보는 다음 목적에만 사용됩니다.

- 멀티플레이 방 생성, 참가, 재연결 처리
- 출발 문서와 도착 문서 기준의 게임 진행
- 현재 문서, 순위, 경과 시간, 경로 공개 등 게임 상태 표시
- 게임 규칙 위반 판정 및 진행 제어

## 4. 로컬 저장

일부 정보는 브라우저의 로컬 저장소(`chrome.storage.local`)에 저장됩니다. 이는 로비 상태, 방 정보, 현재 세션 정보를 유지하고 복원하기 위한 목적입니다.

## 5. 서버 통신

게임 진행을 위해 필요한 정보는 확장 프로그램 서버와 통신할 수 있습니다. 전송되는 정보에는 닉네임, 방 정보, 현재 문서 제목, 진행 상태가 포함될 수 있습니다.

이 정보는 게임 기능 제공 외의 목적으로 판매되거나 광고 목적으로 제3자에게 제공되지 않습니다.

## 6. 웹사이트 접근 범위

이 확장 프로그램은 나무위키 페이지에서만 HUD 표시 및 게임 진행 처리를 위해 동작합니다. 접근 범위는 게임 기능 제공에 필요한 페이지 정보로 제한됩니다.

## 7. 원격 코드

이 확장 프로그램은 원격에서 호스팅된 코드를 다운로드하여 실행하지 않습니다.

## 8. 사용자 선택

사용자는 언제든지 확장 프로그램을 제거할 수 있으며, 저장된 데이터는 방 나가기 또는 확장 프로그램 제거를 통해 정리될 수 있습니다.
